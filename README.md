# AWS Cost Estimator

Industry-grade AWS cost estimation tool with support for 250+ services.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- 250+ AWS Services
- All AWS Regions
- Real-time cost calculations
- Cost comparison across regions
- Export to PDF/CSV/Terraform
- Architecture templates
- AWS Amplify UI

## Development

This is a frontend-only application. Pricing data files should be placed in `public/data/`.

See `ARCHITECTURE.md` for detailed design documentation.
